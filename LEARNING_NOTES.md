## Quick Commands
- Build a single C file:

  ```bash
  gcc -Wall -Wextra file.c -o file
  ./file
  ```

- Build all C files in a folder with gcc (example):

  ```bash
  gcc -Wall -Wextra folder/*.c -o program
  ```
